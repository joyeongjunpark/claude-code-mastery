# 사업기획자 웹 이력서 개발 로드맵

## 📋 프로젝트 개요

LG 전자 공간사업기획팀 재직 중인 2년차 사원의 **인터랙티브 웹 이력서** 개발 프로젝트입니다.

**목표**: 전문성 있고 모던한 디자인의 이력서로 채용담당자의 관심을 끌기

---

## 🎯 주요 기능

| 섹션 | 설명 |
|------|------|
| **헤더** | 프로필 사진, 이름, 직급, 연락처, 소셜 링크 |
| **자기소개** | 핵심 경력 및 강점 요약 |
| **경력사항** | LG 전자 공간사업기획팀 상세 경력 |
| **주요 성과** | 프로젝트별 실적 및 성과 |
| **보유 기술** | 비즈니스 스킬, 데이터 분석 도구, 협업 도구 |
| **교육 및 자격증** | 학력, 자격증, 교육 이력 |
| **포트폴리오** | 주요 프로젝트 및 성과물 (스크린샷/링크) |

---

## 🛠 기술 스택

```
Frontend:
├── HTML5 (시맨틱 마크업)
├── CSS3 (반응형 디자인)
├── JavaScript (인터랙티브 기능)
└── TailwindCSS (유틸리티 기반 스타일링)

Tools:
├── Node.js (개발 환경)
└── npm (패키지 관리)
```

### **Phase 3: 콘텐츠 섹션 구현 (3-4일)**

#### 목표
- 각 섹션별 콘텐츠 구조화 및 스타일링

#### 작업 항목
- [ ] **About 섹션** (자기소개)
  - 핵심 경력 요약
  - 주요 강점 (2-3개)
  - 타임라인 또는 하이라이트 디자인

- [ ] **Experience 섹션** (경력사항)
  - LG 전자 공간사업기획팀 정보
  - 입사일, 직급, 주요 업무 내용
  - 맡은 프로젝트 3-5개 상세 기술

- [ ] **Achievements 섹션** (주요 성과)
  - 프로젝트별 성과 (정량적 지표)
  - 다이어그램 또는 차트 포함
  - 임팩트 강조

- [ ] **Skills 섹션** (보유 기술)
  - 비즈니스 스킬 (전략 기획, 시장 분석 등)
  - 도구 및 소프트웨어 (엑셀, 파워포인트, 태블로 등)
  - 협업 도구 (슬랙, 노션 등)
  - 스킬 레벨 시각화 (프로그레스 바)

- [ ] **Education 섹션** (교육 및 자격증)

  - 관련 자격증 (PMP, CBAP 등 가정)
  - 수료 교육

- [ ] **Portfolio 섹션** (주요 성과물)
  - 프로젝트 카드 (이미지, 제목, 설명)
  - 클릭 시 상세 모달 또는 페이지 링크

#### 산출물
- 모든 섹션 HTML 구조 완성

### 핵심 강점
- **공간사업 전략 기획**: 신규 공간 서비스 및 사업모델 개발
- **데이터 기반 의사결정**: 시장 분석, 소비자 조사, 경쟁사 분석
- **협력사 관리**: 팀간 소통, 프로젝트 관리, 이해관계자 관리

### 주요 경력사항 예시
1. **스마트홈 공간 서비스 사업화** (2023.06 - 2024.03)
   - 신규 서비스 콘셉트 기획 및 사업성 평가
   - 시장조사 및 타겟층 분석 (300명 대상 설문)
   - 경영진 보고 자료 작성 (추진으로 결정)

2. **LG 쇼룸 리뉴얼 프로젝트** (2024.04 - 현재)
   - 기존 쇼룸 방문객 분석 (월 2,000명 → 4,000명 증대)
   - 새로운 공간 콘셉트 3가지 제안 및 선정
   - 시공업체 관리 및 오픈 일정 조율

3. **고객 경험 개선 TF** (2023.12 - 2024.02)
   - 고객 여정 맵핑 및 페인포인트 도출
   - 개선안 3건 제시 → 2건 실행으로 고객만족도 +12% 달성

### 보유 기술
- **비즈니스 스킬**: 사업 기획, 시장 분석, 경영진 보고서 작성, 협상
- **분석 도구**: 엑셀 (고급), 파워BI, 태블로, SQL (기초)
- **협업 도구**: 슬랙, 노션, 서비스나우, SAP

### 자격증
- ADSP (데이터분석 준전문가) - 2023.09
- 비즈니스 기획사 1급 - 2024.01

### 색상 팔레트
```
Primary: #1F2937 (짙은 회색) - 정통성, 신뢰
Accent: #3B82F6 (파란색) - LG 브랜드 이미지
Success: #10B981 (초록색) - 성과, 긍정
Warning: #F59E0B (주황색) - 주의
Gray: #E5E7EB (밝은 회색) - 배경, 구분선
```

### 폰트
- **제목**: Pretendard 700 (한글), Poppins 700 (영문)
- **본문**: Pretendard 400 (한글), Inter 400 (영문)
- **코드**: Fira Code (모노스페이스)

### 간격 및 레이아웃
- 섹션 간격: 80px (모바일), 120px (데스크톱)
- 최대 콘텐츠 폭: 1200px
- 여백: 좌우 16px (모바일), 48px (데스크톱)
